Run a New Scan

To run a new directory scan, use the management command:

python manage.py scan_for_new_images /path/to/photos --pattern="SP-" [--debug] [--skip-existing]

Options:

  • --pattern: Pattern to match in filenames (e.g., SP-2016)
  • --skip-existing: Skip existing images instead of updating them
  • --debug: Enable debug output
  • --slab-images-only: Process images as slab images only
